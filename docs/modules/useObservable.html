useObservable

  • useObservable<T>(initializer: T | (() => T), deps?: DependencyList, options?: UseObservableOptions<T>): T
    • Defined in useObservable.ts:64

    React hooks to make current component reactive to mobx observables, both locally and externally.

    Example:

    // create local store
    let localStore = useObservable({ name: 'jon', description: 'Knows nothing' })

    // use external observable
    useObservable(storeFromEssos)

    // mixed
    let localStore = useObservable({
      name: 'jon',
      get familyName() {
        if (this.name === 'jon') {
          return starkFamily.bastardName // 'snow'
        } else {
          return starkFamily.name // 'stark'
        }
      }
    })

      Plain object or an observable

      Observable initializer

    • deps: DependencyList = []

      Dependency list of initializer. When changed, a new observable will be created
